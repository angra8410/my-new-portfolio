Guía Completa de CSS Grid y Flexbox

10 de marzo de 2024 • 10 min de lectura

CSS Grid y Flexbox son dos herramientas poderosas para crear layouts modernos y responsivos. En esta guía, aprenderás cuándo usar cada uno y cómo combinarlos efectivamente.

¿Grid o Flexbox?

Esta es probablemente la pregunta más común. La respuesta corta: usa ambos. Están diseñados para trabajar juntos y resolver diferentes problemas.

Cuándo usar Flexbox

  • Layouts unidimensionales (filas o columnas)
  • Distribución de espacio entre elementos
  • Alineación de contenido
  • Componentes pequeños (navegación, cards, botones)

Cuándo usar Grid

  • Layouts bidimensionales (filas y columnas simultáneas)
  • Layouts complejos de página completa
  • Posicionamiento preciso de elementos
  • Diseños con áreas nombradas

Flexbox Fundamentals

Flexbox es perfecto para alinear elementos en una dimensión. Aquí están las propiedades más importantes:

Contenedor Flex

.container {
    display: flex;
    
    /* Dirección */
    flex-direction: row | column;
    
    /* Envoltura */
    flex-wrap: wrap | nowrap;
    
    /* Alineación horizontal (eje principal) */
    justify-content: flex-start | center | space-between | space-around;
    
    /* Alineación vertical (eje cruzado) */
    align-items: flex-start | center | stretch;
    
    /* Espaciado */
    gap: 1rem;
}

Items Flex

.item {
    /* Crecimiento */
    flex-grow: 1;
    
    /* Encogimiento */
    flex-shrink: 1;
    
    /* Base */
    flex-basis: 200px;
    
    /* Atajo */
    flex: 1 1 200px; /* grow shrink basis */
    
    /* Alineación individual */
    align-self: flex-start | center | flex-end;
}

CSS Grid Fundamentals

Grid es ideal para layouts bidimensionales complejos:

Contenedor Grid

.container {
    display: grid;
    
    /* Definir columnas */
    grid-template-columns: 1fr 2fr 1fr;
    grid-template-columns: repeat(3, 1fr);
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    
    /* Definir filas */
    grid-template-rows: auto 1fr auto;
    
    /* Espaciado */
    gap: 1rem;
    grid-gap: 1rem 2rem; /* row-gap column-gap */
    
    /* Áreas nombradas */
    grid-template-areas:
        "header header header"
        "sidebar main main"
        "footer footer footer";
}

Items Grid

.item {
    /* Posicionamiento por líneas */
    grid-column: 1 / 3;
    grid-row: 1 / 2;
    
    /* Atajo */
    grid-area: 1 / 1 / 2 / 3; /* row-start / col-start / row-end / col-end */
    
    /* Por área nombrada */
    grid-area: header;
}

Patrones Comunes

Layout Responsivo con Grid

.grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 2rem;
}

/* Sin media queries necesarias! */

Centrado Perfecto

/* Con Flexbox */
.flex-center {
    display: flex;
    justify-content: center;
    align-items: center;
}

/* Con Grid */
.grid-center {
    display: grid;
    place-items: center;
}

Card con Flexbox

.card {
    display: flex;
    flex-direction: column;
    height: 100%;
}

.card-content {
    flex-grow: 1;
}

.card-footer {
    margin-top: auto;
}

Combinando Grid y Flexbox

El verdadero poder viene de combinar ambas tecnologías:

/* Grid para el layout principal */
.page {
    display: grid;
    grid-template-columns: 250px 1fr;
    grid-template-rows: auto 1fr auto;
    gap: 2rem;
    min-height: 100vh;
}

/* Flexbox para la navegación */
.nav {
    display: flex;
    justify-content: space-between;
    align-items: center;
    gap: 1rem;
}

/* Grid para la galería */
.gallery {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
    gap: 1rem;
}

/* Flexbox para cada card */
.card {
    display: flex;
    flex-direction: column;
}

Tips y Trucos

1. Auto-fit vs Auto-fill

  • auto-fit: Colapsa tracks vacíos
  • auto-fill: Mantiene tracks vacíos

2. Minmax para Responsividad

grid-template-columns: repeat(auto-fit, minmax(min(100%, 250px), 1fr));

3. Aspect Ratio con Grid

.square {
    display: grid;
    aspect-ratio: 1 / 1;
    place-items: center;
}

Compatibilidad

Ambas tecnologías tienen excelente soporte en navegadores modernos:

  • Flexbox: >98% de navegadores
  • Grid: >96% de navegadores

Conclusión

CSS Grid y Flexbox no son competidores, son complementarios. Usa Flexbox para componentes y alineación unidimensional, y Grid para layouts complejos bidimensionales. Dominar ambos te convertirá en un desarrollador frontend más eficiente y capaz.

La clave es practicar y experimentar. Cada proyecto te enseñará nuevos patrones y técnicas.
